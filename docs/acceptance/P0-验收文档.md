## 二、阶段目标与范围

### 2.1 阶段目标

打通「提问 → 意图解析 → Me-Form → 预置 SQL/模板 → 答复 → 埋点」，建立安全与可观测底座。

### 2.2 范围内（In Scope）

- [ ] 统一认证 JWT → `UserProfile` 注入
- [ ] LangGraph 主图：`intent_parse` / `reject` / `clarify`
- [ ] Me-Form：`我/本院/全校` 映射与越权拦截
- [ ] 规则防御：注入、敏感词、无关域拒答
- [ ] 埋点表 `log_ai_qa_lifecycle` 与刷盘节点
- [ ] `edu_ai_dw` 首批 **3** 个预置视图
- [ ] 模板化只读 SQL 执行器
- [ ] Golden Set **30** 条冒烟

### 2.3 范围外（Out of Scope，本阶段不验收）

- [ ] Schema RAG / LLM 自动生成 SQL（P1）
- [ ] 文本向量检索 / HYBRID 并行（P2）
- [ ] TIAS 抬头率、图表、多轮续聊（P3）
