## 二、阶段目标与范围

### 2.1 阶段目标

结构化问题从「模板 SQL」升级为 **Schema RAG + LLM 生成 SQL + 静态审计 + 纠错循环**，覆盖教学质量督导主场景。

### 2.2 范围内

- [ ] 指标词典 `dim_metric`（首批 ≥25 条）
- [ ] 离线 `schema_catalog` + 混合检索（BM25+向量）
- [ ] `schema_rag` / `sql_generate` / `sql_audit` / `sql_execute` 完整子图
- [ ] SQL 纠错循环（≤3 次）
- [ ] `edu_ai_dw` 视图扩充至 **8** 个
- [ ] `fusion` 初版（SQL 单分支）
- [ ] Golden Set **50** 条 + SQL 准确率评测

### 2.3 范围外

- [ ] Text RAG、`text_expert`（P2）
- [ ] HYBRID `Send` 并行（P2）
- [ ] Badcase 标注平台（P2）
- [ ] TIAS、图表（P3）
