## 二、阶段目标与范围

### 2.1 目标

- 支持评教/督导 **质性文本** 检索与摘要  
- **HYBRID** 复合问题 SQL + Text **并行** 答复  
- 点踩/失败 SQL → **Badcase 池** → 标注 → Few-Shot 热更新  

### 2.2 范围内

- [ ] 文本切块 pipeline + 向量索引  
- [ ] `text_expert` 节点  
- [ ] `Send` 并行 + `fusion` 交叉验证  
- [ ] `log_ai_qa_badcase_queue` + Worker  
- [ ] Badcase 标注后台 v0  
- [ ] Few-Shot 版本库 + 热更新（24h 内生效）  
- [ ] Golden Set +80（累计约 130），含 HYBRID 压测  
- [ ] `campus_kb`（若未裁剪，需说明）

### 2.3 范围外

- [ ] TIAS 抬头率正式数据（P3，可用 Mock 不计入 P2 否决）  
- [ ] Text-to-Chart（P3）  
- [ ] 多轮 Checkpointer（P3）
